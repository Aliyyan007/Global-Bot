# Contributing to Global Bot

Thank you for considering contributing! This document outlines how to get started and what we expect from contributors.

## How to contribute

1. Fork the repository and create a descriptive branch: `git checkout -b feature/your-feature`.
2. Make small, focused commits. Keep changes readable and well-documented.
3. Run tests or linting locally (if present) before opening a PR.
4. Open a pull request with a clear description of the change and why it's needed.

## Code style

- Follow existing patterns in the codebase. Keep things consistent.
- Prefer clear variable names and avoid large refactors without prior discussion.

## Reporting issues

Use the Discord server for fast triage: https://discord.gg/kaK6JEuRht

If you open an issue on GitHub, include:

- A clear title
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or screenshots

## Security

If you discover a security vulnerability, do not open a public issue. Contact the maintainers via the GitHub profile: https://github.com/aliyyan007
